# repo-cardify

Create stunning GitHub project cards with customizable themes, fonts, and layouts.

## Features

- Customizable themes & color schemes
- Multiple font options
- Flexible layouts
- Export as PNG/SVG

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `pnpm install`
2. Set `GITHUB_TOKEN` in [.env.local](.env.local) to avoid GitHub API rate limits
3. Run the app:
   `pnpm dev`
